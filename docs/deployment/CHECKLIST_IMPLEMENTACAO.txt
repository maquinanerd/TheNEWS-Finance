INSTRUÇÕES PARA USAR:

1. EXECUTAR O SISTEMA:
   $ python main.py
   
   O sistema faz tudo automaticamente:
   - Processa artigos
   - Publica no WordPress
   - Salva JSON com slug
   - Registra tokens com correlação completa

2. PROCURAR UM JSON:
   $ python find_json.py "olimpicos"
   
   Encontra: ai_response_batch_jogos-olimpicos-inverno-2026-filmes-series_20260205.json

3. VER STATUS:
   $ python check_system_status.py
   
   Mostra: JSONs, tokens, posts publicados, tudo correlacionado

4. REORGANIZAR JSONs ANTIGOS (opcional):
   $ python reorganize_jsons.py
   
   Renomeia JSONs antigos para novo padrão
