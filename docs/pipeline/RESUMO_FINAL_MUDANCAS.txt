✅ EXEMPLO PRÁTICO:
────────────────────

Você vê a URL publicada:
  https://www.thefinance.news/jogos-olimpicos-inverno-2026-filmes-series/

Quer encontrar TUDO relacionado:

  1. Procurar JSON:
     $ python find_json.py "olimpicos"
     
     Resultado:
     ✅ ai_response_batch_jogos-olimpicos-inverno-2026-filmes-series_20260205-143521.json

  2. Abrir o arquivo JSON:
     $ Start-Process "debug/ai_response_batch_jogos-olimpicos-inverno-2026-filmes-series_20260205-143521.json"

  3. Verificar tokens e wp_post_id:
     $ Select-String logs/tokens/tokens_2026-02-05.jsonl -Pattern "olimpicos"
     
     Resultado:
     {
       "wp_post_id": 71064,
       "article_title": "Jogos Olímpicos de Inverno 2026: Filmes e Séries para Assistir",
       "source_url": "https://screenrant.com/best-movies-tv-shows-watch-2026-winter-olympics/",
       "metadata": {"slug": "jogos-olimpicos-inverno-2026-filmes-series"}
     }

  4. Acessar post no WordPress:
     https://www.thefinance.news/?p=71064
